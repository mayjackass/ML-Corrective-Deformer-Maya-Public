🎯 Project Overview

A machine learning-based deformer for Autodesk Maya that automatically predicts corrective blendshapes based on skeletal joint poses. This system learns from artist-provided or simulation-derived data to automate the traditionally manual and time-consuming process of creating corrective shapes for character rigs.

🎓 Research Background

This implementation is based on the technical research paper: "Pose-Based Corrective Blendshape Prediction Using Machine Learning in Maya"

Key Contributions

  • ✅ Automated corrective workflow replacing manual sculpting
  • ✅ Real-time prediction embedded in Maya deformation graph
  • ✅ Artist-friendly pipeline for data collection and deployment
  • ✅ Production integration for VFX, animation, and games

Applications

🎬 VFX Production

Automate corrective sculpting, reduce rigging time by 40-60%

🛠️ Technical Specifications

Neural Network Architectures

  • Standard Model: 4 layers [256, 512, 512, 256] - ~604K parameters
  • Compact Model: 3 layers [128, 256, 128] - ~73K parameters
  • Residual Model: 4 residual blocks, 512 hidden - ~2.2M parameters

Performance Targets

Phase 1 (Python)
24 FPS
1,000 vertices
Phase 3 (C++)
60+ FPS

Tech Stack

  • Maya Python API 2.0
  • PyTorch 1.8+
  • NumPy, SciPy
  • TorchScript & ONNX export
  • Future: C++ with LibTorch
