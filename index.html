Overview

This project implements a machine learning-based deformer for Autodesk Maya that automatically predicts corrective blendshapes based on skeletal joint poses. The system learns from artist-provided or simulation-derived data to automate the traditionally manual and time-consuming process of creating corrective shapes for character rigs.

Key Features

  • Complete Phase 1 Python prototype with Maya Python API 2.0
  • Three neural network architectures optimized for different use cases
  • Training pipeline with validation and model export (TorchScript/ONNX)
  • Data collection tools with automatic and manual capture modes
  • Installation system with shelf button and test scene generation
  • Comprehensive documentation and quick-start guides

Requirements

  • Autodesk Maya 2020+ (with Python 3.7+)
  • PyTorch 1.8+
  • NumPy, SciPy for data processing
  • GPU recommended for training (CPU supported)
